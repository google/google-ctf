# Hackceler8 2023 Game

## Steps to run the game:

1. Install prerequisites
```
cd hackceler8-2023
python3 -m venv my_venv
cd game/
source ../my_venv/bin/activate
pip3 install -r requirements.txt
```

2. Run server (optional)

```
cd hackceler8-2023/game
source ../my_venv/bin/activate
python3 server.py
```

3. Run client

```
cd hackceler8-2023/game
source ../my_venv/bin/activate
python3 client.py
```
